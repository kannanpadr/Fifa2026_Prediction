🛡️ Diagnostics Help

  • • Connection State: Shows current mongoose connection ready state (CONNECTED, CONNECTING, DISCONNECTED).
  • • Host / Port: Displays IP or name of the server running MongoDB. Helpful for verifying localhost vs cloud instances.
  • • Database Name: Confirms the active database schema selected (typically fifa2026).
  • • Read/Write Test: Executes a live query test on backend to confirm database transactions are not blocked or slow.
