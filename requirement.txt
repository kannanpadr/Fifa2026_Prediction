FIFA World Cup 2026 Prediction Portal – Requirement Specification
Project Overview

The FIFA World Cup 2026 Prediction Portal is a web-based application that allows football fans worldwide to predict match scores and compete with other users based on the accuracy of their predictions.

The application will be accessible through the internet and support two types of users:

Admin
User

The system will automatically calculate prediction points, generate rankings, and display leaderboards based on actual match results entered by the administrator.

3. Prediction Page
Features

Users can predict the score for each match.

Example:

Brazil vs Argentina

Prediction:

Brazil: 2
Argentina: 1
Prediction Rules
Users can submit predictions only before the first match of the day starts.
Once the first match of the day begins:
All predictions for that day's matches become locked.
Users can:
Create prediction
Edit prediction before deadline
View submitted prediction
Validation Rules
Score must be numeric.
Negative values are not allowed.
Prediction deadline must be enforced automatically using JavaScript and server-side validation.

Admin Module

Prediction Processing

After entering actual results:

System automatically:

Compare all user predictions
Calculate points
Update user scores
Regenerate leaderboard
Update rankings

No manual calculation required.

Point Calculation Rules

Example scoring model:

Exact Score Prediction

Prediction:
Brazil 2 - 1 Argentina

Actual:
Brazil 2 - 1 Argentina

Points: 5

Correct Winner Prediction

Prediction:
Brazil 1 - 0 Argentina

Actual:
Brazil 3 - 1 Argentina

Points: 3

Correct Draw Prediction

Prediction:
1 - 1

Actual:
2 - 2

Points: 3

Wrong Prediction

Points: 0

The point values should be configurable by the administrator.
